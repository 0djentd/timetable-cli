# timetable-cli
## How to use
```
Usage: timetable-cli [OPTIONS] [ACTIVITIES_SELECTOR]...

Options:
  --config TEXT                  [required]
  --db TEXT                      [required]
  --debug
  -d, --global-timedelta TEXT
  -D, --show-date
  -S, --show-status
  -A, --show-activities
  --list-categories
  -c, --columns TEXT
  --table-kwargs TEXT
  --ignore-time-status
  --combine-title-and-variation
  -R, --show-rules
  -r, --rules-list
  -Q, --show-quotes
  -q, --quotes-list
  -W, --watch
  --watch-text TEXT
  --watch-interval INTEGER
  --watch-clear-screen
  --watch-notification
  --watch-notification-cmd TEXT
  --watch-voice
  --watch-voice-cmd TEXT
  --watch-notify-eta TEXT
  --watch-table-selectors TEXT
  --help                         Show this message and exit.
```
